A* Search

Using the path-finding algorithm known as A star search, we can tremendously speed up the time it takes to find a path from point A to point B. In this project, I implement the following AI algorithms: Uniform Search (Dijkstra), A* search, and Weighted A*. This project helps illustrate the usefulness of these algorithms in real life application, such as GPS mapping.

Skills used: Python, OOP, Data Structures, AI algorithms

Mini-Max Search

Using the AI algorithm, Mini-max, the computer will select the best possible move in this grid-based game known as Wumpus-world. The heuristic we used for this algorithm is to select the best possible state in which the computer will have more pieces than the player.

Skills used: Python, OOP, Data Structures, AI algorithms
